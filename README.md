# Books and Brews

Production-ready static site for Books and Brews with public pages, quote/contact forms, client login, portfolio demos, and Cloudflare Pages-ready structure.

## Cloudflare Pages
- Framework: None
- Build command: blank
- Output directory: `/`

## Included
- Home
- Services & Pricing
- Portfolio
- Contact with Turnstile-ready form
- Quote request
- Client login
- Demo pages: Hair, Barber, Beauty

## Backend
Forms post to `/api/contact`. Client login placeholder posts to `/api/client/login`.
